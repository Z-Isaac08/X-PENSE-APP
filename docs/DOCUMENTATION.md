## 🎯 Résumé Rapide

### **L'Application**

- ✅ Gestion budgets (Plafonné / Suivi)
- ✅ Dépenses & Revenus
- ✅ Dashboard avec graphiques
- ✅ Notifications intelligentes
- ✅ Agent IA conversationnel (Groq Llama 3.3 70B)

### **Technologies**

- React 19 + TypeScript + Vite
- Firebase (Auth + Firestore)

## 🚀 Quick Start

```bash
# 1. Clone
git clone https://github.com/Z-Isaac08/X-PENSE-APP.git
cd X-PENSE-APP

# 2. Install
npm install

# 3. Configure .env
cp .env.example .env
# Éditer .env avec tes clés (Firebase, Groq)

# 4. Run
vercel dev

# 5. Deploy (quand prêt)
git add .
git commit -m "feat: app complete"
git push
# Puis importer sur vercel.com
```
